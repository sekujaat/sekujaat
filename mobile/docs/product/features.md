# Product Features – Synce

### Core

- One‑tap random video calls using Agora channels and tokens.  
- Text chat with message history synced across devices via Neon PostgreSQL.  
- Quick search for users and recent contacts to re‑connect easily.  

### Engagement & Safety

- Basic reporting and blocking tools to avoid unwanted contacts.  
- Simple profile with username, avatar, and short bio.  
- Analytics dashboard (Advanced Access) to inspect usage, errors, and call quality stats.  

### Monetization

- Google AdMob banners/interstitials for free users.  
- Premium subscription removing ads and giving priority matching plus extra filters.
